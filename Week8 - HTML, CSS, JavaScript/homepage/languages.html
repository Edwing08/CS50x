Languages and travelling

In my opinion, learning a language is a difficult task. It requires a lot of effort, time and dedication. The first language I decided to learn was English and I gave up many times. Despite this, one day, after a lot of back and forth, I changed my mentality and my way of learning languages. Learning English has opened many doors for me that were closed before. Traveling is easier now that I can speak more than one language, so I will continue to learn languages because of its benefits. this is a list of languages I would like to speak someday.
